🚀 Localhost HTTP Server

A minimalist HTTP/1.1 server built in Rust

Features

  • ✅ Single-threaded, non-blocking I/O
  • ✅ Cross-platform (kqueue/epoll)
  • ✅ HTTP/1.1 request parsing
  • ✅ Keep-alive connections
  • ✅ Static file serving
  • ✅ MIME type detection
  • ✅ Chunked transfer encoding
  • ✅ POST/DELETE methods
  • ✅ File uploads (multipart/form-data)
  • ✅ Cookie and session management
  • ✅ CGI support (Python, Perl, Shell, Ruby, PHP)
  • ✅ Configuration file system (TOML)
  • ✅ Custom error pages
  • ✅ Directory listing
  • ✅ HTTP redirections
  • ✅ Multiple listeners and virtual hosts
  • ✅ Timeout management (5 types)
